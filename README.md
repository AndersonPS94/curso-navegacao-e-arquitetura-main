# 📱 Flutter: Arquitetura e Navegação

Projeto desenvolvido durante o curso **Flutter: Arquitetura e
Navegação** da Alura, com foco em organização de aplicações Flutter,
navegação entre telas e gerenciamento de estado.

------------------------------------------------------------------------

# 📖 Sobre o Projeto

Este projeto foi criado como parte dos estudos sobre **arquitetura e
navegação em Flutter**, abordando conceitos essenciais para a construção
de aplicativos organizados, escaláveis e fáceis de manter.

Durante o desenvolvimento foram aplicadas boas práticas utilizadas no
mercado para separar responsabilidades no código e estruturar melhor a
aplicação.

Entre os principais conceitos explorados estão:

-   Navegação entre telas
-   Rotas nomeadas
-   Passagem de dados entre páginas
-   Organização da arquitetura do projeto
-   Gerenciamento de estado
-   Uso de ChangeNotifier e ValueNotifier
-   Implementação do padrão MVVM

------------------------------------------------------------------------

# 🧠 Conceitos Aprendidos

## Navegação no Flutter

-   Uso do Navigator.push e Navigator.pop
-   Navegação entre páginas
-   Retorno de dados entre telas

## Rotas Nomeadas

-   Estruturação de rotas
-   Organização da navegação da aplicação
-   Manutenção mais simples e escalável

## Gerenciamento de Estado

-   Uso do setState
-   Atualização de UI baseada em estado

## Arquitetura MVVM

Separação da aplicação em camadas:

-   Model → Representa os dados
-   View → Interface do usuário
-   ViewModel → Lógica de negócio

# 🚀 Como Executar o Projeto

## 1️⃣ Clone o repositório

``` bash
git clone https://github.com/AndersonPS94/curso-navegacao-e-arquitetura-main.git
```

## 2️⃣ Entre na pasta

``` bash
cd curso-navegacao-e-arquitetura-main
```

## 3️⃣ Instale as dependências

``` bash
flutter pub get
```

## 4️⃣ Execute o projeto

``` bash
flutter run
```
